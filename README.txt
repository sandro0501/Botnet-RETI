Il seguente progetto utilizza le libreria esterne "psutil" e "tqdm" per il recupero delle informazioni hardware della macchina da parte della Botnet client. 
Per il corretto funzionamento della Botnet client si consiglia dunque l'installazione della suddetta libereria e di tutte le relative dipendenze, tra cui le librerie Microsoft Visual C++ 14.0.

Link al sito di riferimento psutil: https://pypi.org/project/psutil/
Guida all'installazione di psutil: https://github.com/giampaolo/psutil/blob/master/INSTALL.rst
Documentazione ufficiale psutil: https://psutil.readthedocs.io/en/latest/
Link al sito di riferimento tqdm: https://pypi.org/project/tqdm/
Microsoft C++ Build Tools per l'installazione delle librerie Microsoft Visual C++ 14.0: https://visualstudio.microsoft.com/it/visual-cpp-build-tools/
